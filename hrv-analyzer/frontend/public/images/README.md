# Images Directory

This directory contains images for the HRV Analyzer feature cards.

## Required Images

For each feature card, you need 3 images:

### 1. Upload Feature
- `upload-cover.jpg` - Cover image (recommended: 400x600px) - Show file upload interface or data files
- `upload-title.png` - Title image with text "File Upload" (recommended: 300x100px)
- `upload-character.png` - Character/illustration (recommended: 200x300px) - Show upload icon or person uploading

### 2. Analysis Feature
- `analysis-cover.jpg` - Cover image - Show heart rate monitor, ECG graph, or HRV visualization
- `analysis-title.png` - Title image with text "HRV Analysis"
- `analysis-character.png` - Character/illustration - Show heart icon, graph, or medical device

### 3. Charts Feature
- `charts-cover.jpg` - Cover image - Show data visualization, charts, graphs
- `charts-title.png` - Title image with text "Real-time Charts"
- `charts-character.png` - Character/illustration - Show chart icon or data visualization

### 4. Health Feature
- `health-cover.jpg` - Cover image - Show health status, wellness, or stress indicators
- `health-title.png` - Title image with text "Health Status"
- `health-character.png` - Character/illustration - Show health icon, status indicator, or wellness symbol

## Image Suggestions

You can use:
- Free stock photos from Unsplash, Pexels, or Pixabay
- Medical/health themed images
- Heart rate monitor screenshots
- Data visualization graphics
- Wellness and fitness imagery

## Fallback

If images are not found, the component will:
- Use gradient backgrounds based on feature color
- Display text fallbacks for titles
- Hide character images gracefully

## Quick Setup

1. Find or create images matching the descriptions above
2. Name them exactly as listed (case-sensitive)
3. Place them in this `public/images/` directory
4. The app will automatically use them
