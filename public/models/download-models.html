Face API Models Download Guide

Option 1: Direct Download (Recommended)

Visit the official repository and download the models:

https://github.com/justadudewhohacks/face-api.js-models

Option 2: Download via Command Line

If you have Git installed:

git clone https://github.com/justadudewhohacks/face-api.js-models.git temp-models
copy temp-models\weights\*.json public\models\
copy temp-models\weights\*.shard* public\models\
rmdir /s /q temp-models

Required Files

You need these 6 files in the public/models/ directory:

  • tiny_face_detector_model-weights_manifest.json
  • tiny_face_detector_model-shard1
  • face_landmark_68_model-weights_manifest.json
  • face_landmark_68_model-shard1
  • face_expression_model-weights_manifest.json
  • face_expression_model-shard1

Note

The app will try to load models from CDN if local models are not found, but local models provide better performance.
